前缀和数组构造

```c++
		int g[n][m];       
		int sum[n][m];
        for(int i=1;i<=n;i++){
            for(int j=1;j<=m;j++){
                sum[i][j]=sum[i-1][j]+sum[i][j-1]-sum[i-1][j-1]+g[i][j];
            }
        }
```

快速计算 g[a] [b]- g[c] [d]的合

```c++
	int sum(int a, int b, int c, int d) {
        return sum[c][d]-sum[c][b-1]-sum[a-1][d]+sum[a-1][b-1];
    }
```
